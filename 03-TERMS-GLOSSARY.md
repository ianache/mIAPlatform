# Terms and Definition

- AI Agent: es un agente de IA que puede realizar tareas complejas en función a las caracteristicas, skills, tools y personalidad definidas por el usuario.
- Skill: es una capacidad específica que tiene un agente de IA para realizar una tarea determinada.
- Tool: es una herramienta que puede ser utilizada por un agente de IA para realizar una tarea determinada.
- Personality: es la personalidad del agente de IA que puede ser definida por el usuario.
- Knowledge: es el conocimiento que puede ser utilizado por un agente de IA para realizar una tarea determinada.
- LLM: es un modelo de lenguaje grande que puede ser utilizado por un agente de IA para realizar una tarea determinada.
- MCP: es un protocolo de comunicación entre agentes de IA.
- Sandbox: es un entorno seguro donde un agente de IA puede ejecutar herramientas.
- RAG: es una técnica de recuperación de información que puede ser utilizada por un agente de IA para realizar una tarea determinada.
- Vector DB: es una base de datos que puede ser utilizada por un agente de IA para realizar una tarea determinada.
- Prompt: es una instrucción que se le da a un agente de IA para realizar una tarea determinada.
- Context: es el contexto que se le da a un agente de IA para realizar una tarea determinada.
- Memory: es la memoria que puede ser utilizada por un agente de IA para realizar una tarea determinada.
