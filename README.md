## Metodologia

The SDK provides a standalone CLI for autonomous execution:
- gsd-sdk init @prd.md    Bootstrap a project from a PRD
- gsd-sdk auto            Run full autonomous lifecycle
- gsd-sdk run "prompt"    Execute a milestone from text
